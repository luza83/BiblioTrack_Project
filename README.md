# 📚 BiblioTrack App (Frontend)

Frontend application for the BiblioTrack library management system.  
This app allows users to browse, reserve, and manage books through a responsive and intuitive interface.

Built with Vue.js and connected to a .NET REST API.

## ✨ Features

###  Visitor
- Browse and explore available books  
- Reserve books online (ready for pickup after 2 hours)  
- Manage borrowed books  
- Add books to favorites  

###  Admin
- Add, edit, and delete books  
- Manage library inventory and users

###  General
- Search, filter, and pagination  
- Role-based UI (Admin / Visitor views)  
- Responsive design for different screen sizes  

---

## 🧱 Tech Stack

- **Framework:** Vue.js  
- **State Management:** Pinia  
- **API Communication:** Axios  
- **Styling:** (Bootstrap, CSS)  
- **Hosting:** Render  

---

##  Architecture

Frontend (Vue.js) → Backend (.NET API) → Azure SQL Database
